Invalid Credit Card Validation

Form accepts invalid credit card numbers. Missing Luhn algorithm validation.

Location: checkout.js:line 47
Impact: Payment failures, poor UX
AI Suggested Fix 
function validateCard(number) {
  // Luhn algorithm
  let sum = 0;
  for (let i = 0; i < number.length; i++) {
    let digit = parseInt(number[i]);
    if (i % 2 === 0) digit *= 2;
    if (digit > 9) digit -= 9;
    sum += digit;
  }
  return sum % 10 === 0;
}
